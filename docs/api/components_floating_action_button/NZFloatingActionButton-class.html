NZFloatingActionButton

NZFloatingActionButton class

NZFloatingActionButton 是一个多功能的悬浮按钮，支持：

  1. 拖动 (draggable)
  2. 图片背景 (image)
  3. 仅图标或图标加文字 (standard/icon)
  4. 滚动隐藏 (scroll to hide)
