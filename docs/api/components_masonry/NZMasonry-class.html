NZMasonry

NZMasonry class

NZMasonry 是 NezhaUI 提供的专业瀑布流布局组件。

它支持多列等宽不等高的布局，非常适合展示图片、卡片流等内容。 提供基础列表和生成器两种模式，并内置了平滑的进入动画。
